# _Login_Programming

--> Özellikler

Kullanıcı adı ve şifre kontrolü

Maksimum 3 giriş hakkı

Her hatalı girişten sonra kalan hakların gösterilmesi

Giriş başarılı olduğunda programın sonlandırılması

Haklar bittiğinde girişin engellenmesi

--> Kullanılan Konular

while döngüsü

if / else koşulları

const ve var kullanımı

Console.ReadLine() / Console.WriteLine()

String karşılaştırmaları

--> Nasıl Çalışır?

Program kullanıcıdan username ve password ister.

Girilen bilgiler önceden tanımlı bilgilerle karşılaştırılır.

Bilgiler doğruysa:

Giriş başarılı mesajı gösterilir.

Program sonlanır.

Bilgiler yanlışsa:

Giriş hakkı 1 azalır.

Kalan hak kullanıcıya bildirilir.

Haklar biterse:

Giriş başarısız mesajı gösterilir.

Program kapanır.
